QuizJS is a simple web-based quiz application created using **JavaScript**, **HTML**, and **CSS**.  
The project allows users to answer single-choice questions and receive immediate feedback.

## Features
- User dashboard
- User session persistence 
- Quiz recommendations
- Quiz categories loaded from an external API
- Difficulty selection (easy / medium / hard)
- Question count selection
- Two game modes:
  - **Standard mode**
  - **Speedrun mode**
- Quiz history tracking
- Responsive design (mobile & desktop)

## Technologies Used
- HTML  
- CSS  
- JavaScript (Vanilla JS)  
- External Quiz API  

## API
The application uses an external API to fetch quiz categories and questions dynamically based on:
- Selected category
- Difficulty level
- Number of questions

## Game Modes
### Standard Mode
Answer questions at your own pace without time pressure.

### Speedrun Mode
Answer questions as fast as possible with time limit.


##  Project Purpose
This project was created for learning purposes to practice:
- JavaScript logic and state management
- Working with APIs
- Session storage
- UI/UX design
- Responsive web development
